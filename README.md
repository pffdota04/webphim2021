# Trang web xem phim trực tuyến K-Phim
[Live Demo](https://kphim.xyz/)

(Để chạy trang web cần cả 2 phần frontend và backend chạy cùng lúc!)

## Cài đặt
1. Clone this repo
2. Cài đặt package
```bash
npm i
```
3. Khởi động chương trình
```bash
npm start
```
4. Web đang chạy trên ```http://localhost:3000/```
